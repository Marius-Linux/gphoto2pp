void gphoto2pp::CameraFileWrapper::adjustNameForMimeType ( )

Adjusts the file name to match the detected MIME type.

Recognized MIME Types:
  • RAW -> raw
  • JPEG -> jpg
  • PNG -> png
  • PPM -> ppm
  • PGM -> pgm
  • PNM -> pnm
  • TIFF -> tif
  • WAV -> wav
  • BMP -> bmp
  • AVI -> avi
  • CRW -> crw
  • CR2 -> cr2
